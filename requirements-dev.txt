#   保证三条 CI job 使用同一套 Python 工具版本，避免「本地 lint 过、CI lint 挂」。
#
# 【不在本文件中的依赖】
#   Ansible Galaxy collections → ansible/requirements.yml
#   gitleaks                   → install-deps.sh profile gitleaks（二进制）
#   shellcheck / docker        → 系统或 Action，非 pip
#
# 【安装方式（由 install-deps.sh 调用）】
