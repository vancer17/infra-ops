## 一、结论

**Hub 独占安全组规则已按家用动态 IP 场景收口；WG 与 SSH 主路径验收通过。**

80/443 在 WG 内表现为 `connection refused`（**无 Nginx 监听**），与安全组已放行 `10.200.0.0/16 → 443` 不矛盾。部署 Hub Nginx 前须补 **TCP 80 ← 10.200.0.0/16**。
